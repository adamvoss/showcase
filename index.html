Libraries

  1. ArduinoDUE_OBD_FreeRunningCAN

    Website

    Free-running CAN and OBDII data acquisition

  2. J1850 VPW

    Website

    J1850 VPW protocol for M2

  3. LIN

    Website

    LIN bus functionality for M2 on 2 channels

  4. M2_12VIO

    Website

    Provides 6x12V output drivers with over current protection for the output drivers and 6x12V analogue input circuits, plus support for the DUE on chip temperature.

  5. M2_SD_HSCMI

    Website

    Gives access to HSMCI (High Speed MultiMedia Card Interface) on M2.

  6. OBD9141

    Website

    A class to read an ISO 9141-2 port found in OBD-II ports.

  7. Single-Wire CAN (MCP2515)

    Website

    Gives M2 single-wire CAN (GMLAN) functionality. Uses external MCP2515 transceiver connected to SAM3X via SPI. Library currently includes an example of sending a SW CAN message.

  8. due_can

    Website

    Object oriented canbus library for Arduino Due compatible boards

  9. due_wire

    Website

    An alternative I2C library for Due with DMA support

  10. extEEPROM

    Website

    Can be used to program the M2's EEPROM

Projects

  1. CanCat

    Website

    Swiss army knife of Controller Area Networks (CAN) often used in cars and building automation that can run on the Macchina M2

  2. M2RET

    Website

    Reverse Engineering Tool running on Macchina M2 Hardware

  3. SavvyCAN

    Website

    QT-based cross platform CANbus tool. May be used with data collected via the Macchina M2 interface board.
